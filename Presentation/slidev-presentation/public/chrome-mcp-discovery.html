Research Results: Browser MCP for Claude Code

Great news! This already exists. Google released Chrome DevTools MCP in public preview that does exactly what you're asking for.

1. Chrome DevTools MCP (Official from Google)

Capabilities:

Feature Description
Console reading Read browser console logs, errors, warnings
Error analysis AI sees runtime errors and can trace them
JavaScript execution Execute JS in browser context for debugging
Network inspection Analyze requests, CORS errors, API failures
DOM inspection Check page structure, CSS, layout
Screenshots Visual verification of changes
Performance traces Record and analyze performance data

Installation for Claude Code:

claude mcp add chrome-devtools -- npx chrome-devtools-mcp@latest

Answering Your Questions

  1. Can Claude Code see runtime errors like your example?
    YES - Chrome DevTools MCP reads console.error, uncaught exceptions, and network failures.
  2. Can Claude Code query developer console?
    YES - The MCP server exposes console.log, console.warn, console.error output
  3. Can Claude Code print debug messages?
    YES - Via JavaScript execution in browser context
  4. Deployment Architecture?
    Recommended: Use the official Chrome DevTools MCP (NPX package via MCP protocol)
